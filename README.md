# 🎬 Film Takip Uygulaması

C# Console Application kullanılarak geliştirdiğim basit bir film takip sistemi.

## 🚀 Özellikler
- Film ekleme
- Film listeleme
- İzlenen filmleri işaretleme
- Film silme
- Filmlere puan verme
 
## 🛠 Kullanılan Konular
- C#
- List
- Class / Object
- Constructor
- Metotlar
- Switch-Case
- For Döngüsü
- If-Else
- TryParse

## 📌 Proje Amacı
Bu projeyi C# pratiği yapmak, object mantığını geliştirmek ve console uygulamaları üzerinde çalışmak için geliştirdim.

## 📷 Uygulama
Kullanıcı menü üzerinden işlemleri seçerek filmleri yönetebilir ve puanlayabilir.
